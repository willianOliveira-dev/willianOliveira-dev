##  Sobre Mim

[cite_start]Sou um **Desenvolvedor Fullstack Especialista Backend** [cite: 2][cite_start], focado em projetar e entregar APIs escaláveis e sistemas distribuídos[cite: 6]. [cite_start]Minha abordagem de engenharia prioriza *API-first design*, processamento assíncrono orientado a eventos e arquitetura modular[cite: 7], garantindo que o software não apenas funcione, mas suporte alto tráfego com estabilidade.

[cite_start]Moro no Rio de Janeiro  [cite_start]e atuo cobrindo todo o ciclo de vida da aplicação: desde a modelagem de bancos de dados robustos e camadas de autenticação seguras, até a entrega de interfaces web e mobile de alta performance[cite: 8].

**Onde eu me destaco:**
- [cite_start]⚙️ **Arquitetura & Escalabilidade:** Desenvolvimento de pipelines orientados a microsserviços com message-driven architecture (RabbitMQ)[cite: 7].
- [cite_start]🧠 **Inteligência Artificial:** Integração nativa de sistemas com IA Generativa (Groq, Gemini, Vercel AI SDK) para automação e análise de dados em tempo real[cite: 7, 11].
- [cite_start]🛡️ **Qualidade & Segurança:** Foco absoluto em type safety, arquitetura limpa (Clean Architecture) [cite: 8] [cite_start]e segurança end-to-end (JWT, HTTP-only Cookies)[cite: 11].
